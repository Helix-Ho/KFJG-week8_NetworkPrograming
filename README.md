# Study Notes Repository

CS:APP 11장 네트워크 프로그래밍을 학습하며 떠오른 키워드와 질문을 팀원끼리 정리하는 저장소입니다.

- 각자 본인 폴더의 `keywords.md`, `questions.md`에 작성합니다.
- 모든 파일은 CS:APP 11장의 세부 챕터 기준으로 같은 구조를 사용하며, 챕터 제목은 눈에 잘 들어오도록 배너 형태로 구분했습니다.
- 각 챕터마다 더미 항목 3개씩 넣어 두었고, 항목 사이는 가로선으로 구분했습니다. 부족하면 같은 블록을 복사해서 이어서 쓰면 됩니다.
- 숫자 카운팅 없이 가볍게 기록할 수 있도록 구성했습니다.

## Folder Structure

- `Kho/`
- `Kim/`
- `Yoo/`
- `Cho/`

## Writing Guide

- 키워드는 가능하면 한 항목에 한 주제씩 적습니다.
- 질문은 막히는 지점, 토론거리, 해결 후 회고까지 자유롭게 적습니다.
- 링크, 예시, 코드, 추가 메모는 각 항목 아래에 자유롭게 덧붙여도 됩니다.
- 공통 파일명과 챕터 구조를 유지하면 나중에 취합하고 후가공하기 편합니다.
