## Writing Guide

- 키워드는 가능하면 한 항목에 한 주제씩 적습니다.
- 질문은 막히는 지점, 토론거리, 해결 후 회고까지 자유롭게 적습니다.
- 링크, 예시, 코드, 추가 메모는 각 항목 아래에 자유롭게 덧붙여도 됩니다.
- 공통 파일명과 챕터 구조를 유지하면 나중에 취합하고 후가공하기 편합니다.
